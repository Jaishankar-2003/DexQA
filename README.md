# DexQA Website

A modern, premium-quality website for DexQA - a software quality assurance company.

## Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **TypeScript**
- **Tailwind CSS**

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
dexQA/
├── app/
│   ├── layout.tsx       # Root layout with fonts
│   ├── page.tsx         # Home page
│   └── globals.css      # Global styles
├── components/
│   ├── Header.tsx       # Navigation header
│   ├── Hero.tsx         # Hero section
│   ├── TrustMetrics.tsx # Trust metrics strip
│   ├── AboutUs.tsx      # About section
│   ├── Services.tsx     # Services grid
│   ├── QAPackages.tsx   # QA packages
│   ├── WhyDexQA.tsx     # Benefits section
│   ├── ContactCTA.tsx   # Contact form
│   └── Footer.tsx       # Footer
└── package.json
```

## Design System

- **Primary Color**: #0A1A2F (Deep Navy)
- **Accent Color**: #007BFF (Electric Blue)
- **Secondary Color**: #00C7B7 (Teal)
- **Typography**: Poppins/Montserrat (headings), Inter/Roboto (body)

## Build for Production

```bash
npm run build
npm start
```
